# Translation Error Taxonomy

The app uses six main categories:

1. Accuracy
2. Terminology
3. Fluency
4. Style/Register
5. Locale/Cultural
6. Formatting

## Severity Levels

- Minor: small issue; does not seriously affect meaning.
- Major: affects meaning, clarity, or quality.
- Critical: makes the translation seriously wrong or unusable.

## Human-in-the-loop rule

AI output is draft feedback only. Teacher review is required before feedback is used for assessment.
