# Assessment Rubric

Suggested weighted rubric:

| Dimension | Weight | Description |
|---|---:|---|
| Accuracy | 35% | Meaning is preserved accurately. |
| Fluency | 20% | The translation is grammatically correct and natural. |
| Terminology | 20% | Key terms are translated correctly and consistently. |
| Style/Register | 15% | The translation matches the appropriate tone and genre. |
| Mechanics/Formatting | 10% | Spelling, punctuation, layout, and formatting are acceptable. |
